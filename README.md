# 🎓 DDU Alumni Website

A responsive alumni management website developed to strengthen the connection between students and alumni of Dharmsinh Desai University (DDU). The project provides a clean interface for sharing information, exploring alumni profiles, and improving communication within the university community.

---

## ✨ Features

* Responsive user interface
* Home page with navigation
* Alumni information display
* Contact section
* User-friendly design using Bootstrap
* Mobile-friendly layout

---

## 🛠️ Tech Stack

### Frontend

* HTML5
* CSS3
* JavaScript

## 🚀 Getting Started

1. Clone the repository

```bash
git clone https://github.com/Ronit-Vyas/DDU-Alumni-Website.git
```

2. Open the project folder.

3. Launch `index.html` in your preferred web browser.

No additional installation is required.
